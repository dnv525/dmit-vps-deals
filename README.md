## 几个实际用下来的感受

机器本身没什么好说的，AMD EPYC 全系标配，企业级 SSD，硬件层面不是看点。

真正的差距在线路。用 Pro 系列的话，国内电信高峰期晚上九点刷视频，速度基本不受影响，这个体验在同价位里确实难找到对手。Eyeball 系列用下来，大部分时间也挺稳，偶尔高峰期会感觉比 Pro 稍微有点延迟波动，但对于普通使用来说够用了。

还有退款这块。3 天内流量用不超过 30 GB 可以全额退（退到账户余额免手续费），30 天内可以按剩余价值部分退。不确定适不适合的话，先买最小套餐跑几天测一下线路，真不合适还能退，风险不大。

## FAQ

**Q：DMIT 优惠码一定要手动输入吗，会自动应用吗？**

A：不会自动应用，必须在购物车页面手动粘贴进 "Apply Promo Code" 输入框，点 Validate Code 确认。建议复制粘贴，不要手打，否则可能因为大小写或符号错误导致无效。

**Q：月付能用优惠码吗？**

A：大多数优惠码要求季付（3 个月）及以上才能激活。东京 T1 有一个月付专属码 `2025-TYO-T1-HI-GSL-MONTHLY-10OFF`，是 9 折，适合先月付测试。其他系列月付基本是原价。

**Q：用了优惠码之后续费还是折扣价吗？**

A：标注「循环（Recurring）」的优惠码续费一样打折，不存在「首年优惠坑」。这是 DMIT 优惠码里比较良心的一点，买之前确认一下码的备注里有没有 Recurring 字样。

**Q：DMIT 的 IP 被 GFW 封了怎么办？**

A：每 15 天可以免费申请换一次 IP，之后 5 美元一次，通过工单申请。这个政策写在官方条款里，有明确规则，不是口头承诺。

**Q：现在最划算的入手方式是什么？**

A：如果预算有限、先试水：洛杉矶 LAX.T1.WEE 或 LAX.Pro.WEE，$36.9/年，年付价格已经很低了；如果想要中国大陆优化、又要性价比：洛杉矶 LAX.EB 系列叠加 8 折循环码；如果对延迟要求高：香港 Premium，叠加 `202510_HKG_TYO_PRO_20OFF_RECURRING`；香港 T1 年付叠加 45% 优惠码是最大力度的综合优惠。
